# 体重日记 2.0 免费赞助版

《体重日记》是一款纯本地存储的体重记录小工具。它不会上传你的体重、日记或任何健康数据，也不需要登录账号。

## 下载文件

- `weight-diary-windows-x64.zip`：Windows 电脑版
- `weight-diary-android-debug.apk`：Android 安卓版

## 本版亮点

- 所有核心功能永久免费开放。
- 支持每日体重和日记记录。
- 支持本月趋势和全部历史趋势。
- 支持目标体重线。
- 支持 CSV / JSON 导出备份。
- 新增“关于 / 赞助作者”窗口，赞助完全自愿。

## 说明

Android 版本当前为 APK 安装包。首次安装时，部分手机可能提示“未知来源应用”，允许本次安装即可。
