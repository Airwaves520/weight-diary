# 《体重日记》官网发布指南

这个文件夹就是官网成品：`D:\体重日记\官网`

## 已包含内容

- `index.html`：官网首页
- `styles.css`：页面样式
- `assets/images`：软件图标和截图
- `downloads`：Windows 和 Android 下载包
- `robots.txt`：搜索引擎抓取规则
- `sitemap.xml`：站点地图

## 当前发布地址

默认使用 GitHub Pages：

```text
https://airwaves520.github.io/weight-diary/
```

如果以后你买了自己的域名，再把 `robots.txt`、`sitemap.xml` 和 `index.html` 里的 GitHub Pages 地址替换成真实域名即可。

## 怎么让别人能检索到

1. 把本仓库推送到 GitHub。
2. 在仓库 Settings -> Pages 中启用 GitHub Pages。
3. Source 选择 `Deploy from a branch`。
4. Branch 选择 `main`，Folder 选择 `/docs`。
5. 等 GitHub 生成网站后，到百度搜索资源平台、Google Search Console、Bing Webmaster Tools 提交首页和 `sitemap.xml`。
6. 在朋友圈、小红书、B站动态、GitHub/Gitee 项目页都放上官网链接。外部链接越多，越容易被搜索引擎发现。
